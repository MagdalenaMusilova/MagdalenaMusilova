# Software Engineering Portfolio

Backend-focused software engineering student at FIT CTU in Prague, mainly working with C# and .NET.

The projects below include work from my studies and personal projects.
## Projects

### [Job Application Tracker](https://github.com/MagdalenaMusilova/JobApplicationTracker)

Full-stack web application for managing job applications, recruitment stages, interviews, tasks, resumes, and AI-based job matching.

**Built with:** .NET 8, ASP.NET Core, PostgreSQL, Entity Framework Core, Next.js, TypeScript, React, Tailwind CSS, JWT, OpenAI

**Highlights:**

- REST API with authentication and refresh tokens
- PostgreSQL database with EF Core migrations
- Resume upload and PDF parsing
- AI-based resume-to-job matching
- Dashboard statistics and application tracking
- Backend and frontend tests

---

### [FolderSynchronizer](https://github.com/MagdalenaMusilova/FolderSynchronizer)

Console application for one-way folder synchronization from a source folder to a replica folder.

**Built with:** C#, .NET 9, NUnit, Serilog, System.IO.Abstractions

**Highlights:**

- Periodic or one-time synchronization
- File copying, updating, and deletion
- Console and file logging
- Testable file-system abstraction
- Unit tests and benchmark results

---

### [Sequential Quatromin Placement Solver](https://github.com/MagdalenaMusilova/Solver)

C++ solver for a weighted board-placement optimization problem, implemented with multiple execution strategies.

**Built with:** C++20, CMake, OpenMP, MPI, Python benchmarking

**Highlights:**

- Sequential solver
- Task-parallel and data-parallel versions
- MPI-based distributed version
- Input parsing and benchmark support
- Focus on algorithm design and performance comparison

---

### [ASCII Art Converter](https://github.com/MagdalenaMusilova/ASCIIArt)

Scala command-line application that converts images into ASCII art.

**Built with:** Scala, sbt, ScalaTest

**Highlights:**

- Image loading and random image generation
- Console and file output
- Custom ASCII shading tables
- Filters such as rotate, scale, invert, flip, brightness, and ratio correction
- Modular design with tests
